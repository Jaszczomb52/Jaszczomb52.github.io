Game made in Unity

As a mainly C# developer, Unity seems to be perfect. It also offers quite a lot examples etc. and is quite simple if compared to for example Unreal Engine. It also automatically do all the problematic and boring android stuff, like permissions, sensors handling, deployment for this platform etc.

Some details about this project

This project was made for a simple reason. I really don't like Android and Java! Honestly, Java for PC is not that bad, I actually had a short journey with learning Java, but the Android platform because of what it has to be, I mean here safety of user data and the fact that we store a lot of private information on one little device, is very hard to work with, at least for me. I like my window applications for, well... Windows. (MAUI apps are actually cool too) But long story short, when you have to create a project of application for a mobile device to pass subject at your engineering studies and you absolutely hate android, it's basically what's left. And that's how this game was created, as a way to pass subject. It was a lot of fun, everything is created by me or is free to use, for example music, and textures.
