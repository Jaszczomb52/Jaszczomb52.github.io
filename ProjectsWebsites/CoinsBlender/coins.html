Bag, rope, chest and lamp

In this project i wanted to create bag of coins. I used physics simulation and pressure stuff in Blender to create the bag that seems at least a little bit beliveable to be real. The rope was made using Bezier Curve and some modifiers. Lamp and chest were from website that i can't remember now, but were free to use, I only added light to the lamp.

Blender view

And that's how it looks in the Blender, it's nothing amazing obviously, but i had a lot of fun discovering new things in software and modifying light and camera settings to create the render i want.
