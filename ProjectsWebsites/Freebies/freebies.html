Check what's available for free!

Old deals graying out

3 Months old or deals with publish date older than the moment you use the app will gray out automatically! You can always disable it using checkbox at the bottom of the window!

Filters!

Use filter to find only what you want! You can filter out only games for some consoles or from specific platform. You can also choose to find only games.

Choose the order

If you want you can also order by something. You can choose from 4 data types. By how much is the deal worth, by title, by published or end date! All of that in ascending or descending order!

Some details about this project

This application is using an API from gamerpower.com website. It's basically GUI for it. It was made using forms project in C# and has custom made cards to show info about the discount!


  • If you want to check how i made it here's a link to the project on github: https://github.com/Jaszczomb52/GamesDiscountsApp

  • If you want to visit site that's hosting the API, here's a link: https://www.gamerpower.com/

    • As you'll probably notice, the site already has a GUI for that purpose on their website, so you may ask, why even bother? I just wanted to try new things, like creating managers, factories and other strategies in the code. I tried to learn and practice more OOP with proper code that will not turn itself into nightmare in short time. I tried to use interfaces and follow different principles seeking good solutions and their implementations. It was more like a playground to get into thinking the proper way after quite a long break from developing.
